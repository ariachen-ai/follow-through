# Voice Profile

This file captures the user's messaging style. It starts with an example profile and evolves as the skill learns from your actual sent messages.

## How This Works
1. First use: the skill drafts messages using the example profile below as a starting point
2. You edit or rewrite the drafts, then send your own version
3. The skill compares drafts vs actual and updates this file with your patterns
4. Each event, the drafts get closer to your real voice
5. Eventually, replace the example section entirely with your own

## Example Voice Profile (replace with your own)

> The following is a real voice profile learned after one hackathon event. Use it as a reference for what a filled-out profile looks like, then replace it with your own patterns.

### Collab Messages (people with specific follow-up)
- References actual conversation topics — specific projects, advice given, shared interests
- Offers value first — shares events, offers office tours, mentions her own projects they could test
- Shows genuine enthusiasm — "that was the highlight of the event for me"
- Adds humor/personality — ":D", "I'll probably force you to be my test user"
- Mentions future touchpoints — upcoming events, trips, conferences
- Specific collab asks — "Let me know if there's any opportunity you see we could collab on this"

### Stay-in-Touch Messages (casual connections)
- Short and sweet — 1-2 sentences max
- Always offers value — "I'll share any fun events I come across"
- Uses "hii" for casual contacts, "Hey" for others
- Emoji: 😊 for warm, :D for playful — never 🙌
- Sometimes adds a personal touch — e.g. "I am also working on an agent product... would love to learn more from you"

### Tone & Style
- Warm, genuine, enthusiastic
- Uses ":D", "haha", "hii" naturally
- Lowercase casual ("Lets" not "Let's" sometimes)
- 😊 is the go-to warm emoji
- Longer messages for people she cares about, short for casual
- Never stiff or formal

### Language Rules
- Writes in Chinese to Chinese-speaking contacts — not an English template translated
- Chinese style: casual, warm, uses ！and 啊
- Example: "hii [name] 很喜欢你们那天做的产品！我们以后多多交流啊，有好的活动我也和你分享！"

### Message Structure — Collab
1. Greeting + event reference
2. Specific thing she loved about THEIR work/conversation
3. Personal connection (her own related project/interest)
4. Specific collab ask or value offer
5. Stay in touch + coffee invite (if local)

### Message Structure — Stay-in-Touch
1. "Hey/hii [Name]!"
2. "Great meeting/connecting at [Event]"
3. Optional: one personal touch
4. "Let's stay in touch, I'll share any fun events I come across"
5. Emoji: 😊 or :D

### What User Changed from Drafts
- Dropped 🙌 emoji → uses 😊 or :D instead
- Dropped generic openers like "Such a fun event" → replaced with specific references
- Added "I'll share any fun events I come across" (value offer — wasn't in drafts)
- For Chinese contacts: rewrote entirely in Chinese instead of English
- Changed vague project descriptions to what the person actually does

### What User Kept from Drafts
- Humor lines — "7x hackathon winner is insane haha" kept verbatim
- General warm message structure
- Coffee invite phrasing for local contacts
